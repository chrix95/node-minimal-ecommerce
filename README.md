### Instruction to use
- Clone the project 
``` bash
git clone https://github.com/chrix95/node-ecommerce
```
- Change to the `node-ecommerce` directory and run 
``` bash
yarn install
```
- Setup your `.env` variables
``` txt
MONGO_URL=
PORT=3000
PASSWORD_SECRET=
JWT_SECRET=
STRIPE_KEY=
```
- Serve the project on the defined port using:
```bash
yarn start
```
